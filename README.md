# Medicare Drug Spending Analysis

Analysis of Medicare Part D drug spending data (2019-2023) using Python.

## Key Findings
- Brand drugs cost **5x more** than generic drugs on average
- Top spending drug: **Eliquis** at $18.3B in 2023
- Potential savings if switched to generic: **$199.3B**

## Analysis
- Brand vs Generic cost comparison
- Top 10 drugs by Medicare spending
- Cost saving opportunities
- Spending trends 2019-2023
